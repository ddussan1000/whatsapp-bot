# whatsapp-bot (monorepo)

Proyecto separado en dos apps:

- `backend/`: API de WhatsApp Bot con Bun + Hono.
- `dashboard/`: Frontend React + Vite para administración.

## Backend

```bash
cd backend
bun install
cp .env.example .env
bun run dev
```

## Dashboard

```bash
cd dashboard
bun install
cp .env.example .env
bun run dev
```
